This mod introduces an expansive palette of 75 new blood colours to RimWorld, This mod is purely cosmetic and won't change anything about the base game.

![img](https://imgur.com/krl2et8.png)
Adds new colours to
- Dirtmole - Grey
- Genie - Dark Red
- PigSkin - Dark Red
- Impid - Orange
- Yttakin - Dark Red
- Highmate - White
